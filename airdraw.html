User Guide

📐 Position your hand properly:
• Keep your hand 30-60cm from the camera
• Ensure good lighting on your hand
• Spread fingers clearly for detection

👆 Index Finger (Draw Mode):
• Extend only your index finger
• Keep other fingers closed
• Move finger to draw on canvas

✌️ Peace Sign (Place Shapes):
• Extend index and middle fingers
• Keep other fingers closed
• Use to place shapes after sizing

✋ Open Hand (Eraser/Drag):
• Spread all 5 fingers openly
• Use to erase or drag objects
• Move hand over area to erase

💡 Tips for better detection:
• Avoid cluttered backgrounds
• Keep hand steady when gesturing
• Ensure camera is at eye level
🎯 Activating Eraser Mode:
• Click the ERASE button in toolbar
• Or use ✋ open hand gesture

👋 Eraser Hand Position:
• Spread all 5 fingers openly
• Hold hand 30-50cm from camera
• Ensure palm faces camera directly
• Keep fingers spread wide for detection

🔄 Using the Eraser:
• Move your open hand over areas to erase
• The eraser ring shows active area
• Move slowly for precise erasing
• Large hand movements erase larger areas

💡 Eraser Tips:
• Good lighting improves detection
• Keep hand steady while erasing
• Use index finger for precise control
• Switch back to DRAW mode when done
🎨 Activating Shapes Mode:
• Click the SHAPE button in toolbar
• Select shape type (Square, Rectangle, Triangle, Circle)

👆 Drawing Shapes:
• Use ☝️ index finger to start shape
• Drag finger to size the shape
• Shape preview appears as you drag

✌️ Placing Shapes:
• Use ✌️ peace sign to place shape
• Shape becomes permanent on canvas
• Can drag placed shapes with ✋

✋ Moving Shapes:
• Use ✋ open hand to drag shapes
• Hover over shape to grab it
• Move to new position
• Release to place

💡 Shape Tips:
• Start with index finger at desired corner
• Drag diagonally for rectangles
• Use peace sign quickly after sizing
🎨 Activating Text Mode:
• Click the TEXT button in toolbar
• Text input panel appears

⌨️ Adding Text:
• Type your text in the input field
• Click "+ Add Text" button
• Text appears on canvas
• Maximum 50 characters per text

👆 Positioning Text:
• Use ☝️ index finger to position
• Click to place text at location
• Double-click text to edit/delete

✋ Moving Text:
• Use ✋ open hand to drag text
• Hover over text to grab it
• Move to new position
• Release to place

🗑️ Deleting Text:
• Double-click text to show options
• Click "Delete Text" button
• Or select and use delete option

💡 Text Tips: • Keep text short for readability
• Use contrasting colors
• Position away from drawings
🧹 Clearing Canvas:
• Click the trash icon in toolbar
• Entire canvas will be cleared
• Cannot undo clear action

📏 Adjusting Thickness:
• Use + button to increase brush size
• Use - button to decrease brush size
• Size indicator shows current thickness

💾 Saving to Gallery:
• Click Save button in sidebar
• Artwork saved to local gallery
• Access anytime via Gallery button

🖼️ Using Gallery:
• Click Gallery button in sidebar
• View all saved artwork
• Select multiple for batch operations
• Delete or export selected items

📤 Export Options:
• 📄 PDF with Text: Extract text as typable content with image preview
• 📊 PowerPoint: Export as PPTX slideshow presentation
• 📦 ZIP Archive: Download all images as compressed ZIP file
• 🖼️ Image: Export as PNG/JPG image file

💡 Tool Tips:
• Save frequently to avoid loss
• Use export for sharing work
• Gallery works offline
• Export formats preserve quality
